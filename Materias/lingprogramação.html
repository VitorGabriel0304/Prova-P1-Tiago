Linguagem de Programação

O que você vai aprender?

A disciplina de Linguagem de Programação tem como objetivo introduzir os fundamentos das linguagens utilizadas no desenvolvimento de software e aplicações. Durante o curso, você aprenderá:

  • Conceitos de Linguagens: Diferenças entre linguagens de alto e baixo nível e suas respectivas aplicações.
  • Estruturas de Controle: Como usar condicionais, laços de repetição e funções para controlar o fluxo de um programa.
  • Paradigmas de Programação: Estudo de programação estruturada, orientada a objetos e funcional.
  • Manipulação de Dados: Trabalhando com entrada e saída de dados, variáveis e tipos de dados.
  • Boas Práticas de Código: Estilo de código limpo, legibilidade e organização de projetos.
  • Introdução à Depuração: Como identificar e corrigir erros em seus programas.

A matéria explora uma linguagem específica amplamente utilizada no mercado (como Python, Java ou C#), proporcionando aos alunos as habilidades necessárias para aplicar conceitos de programação em problemas reais.
